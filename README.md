# Garry's Mod Deathmatch

This is a port of Garry's Mod Deathmatch, which can be found in "Garry's Mod 10"

## Weapons
The current weapons, that do work inside of this Gamemode are:

- Crowbar
- Pistol
- SMG
- Shotgun

Current weapons, that currently don't work:

- Tripmine
- Flamethrower

Current weapons, that currently haven't been tested:

- Egon
